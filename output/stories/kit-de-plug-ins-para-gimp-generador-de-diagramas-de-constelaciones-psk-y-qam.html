Constelaciones

Este conjuto de plug-ins para gimp escritos en python permiten generar diagramas de constelaciones para las técnicas de modulación PSK Y QAM como los usados en esta web.

  • generator-psk.py
  • generator-qam-c-2.py
  • generator-qam-c-m.py
  • generator-qam-rp.py
  • generator-qam-ro.py
  • generator-qam-ro2.py

LICENCIA: COPYING GNU GPL

INSTALACIÓN:

Requiere tener instalado: gimp, python y gimp-python (Python support and plugins for The GIMP)

1) Copiar los archivos de extención .py en el directorio $HOME/.gimpx-y/plug-ins/ (donde x-y es la versión de gimp, por ejemplo 2-2 para gimp 2.2 ).

2) Darle permiso de ejecución a los archivos generator-* : chmod +x generator-*.

3) Ejecutar gimp.
